Que es un selector?
Es una version en formato STRING del DOM

Los selectors se utilizan para crear LOCATORS

Cuales son los tipos de selectores?
-CSS
-CLASS
-NAME
-ID
-TEXT
-XPATH
-etc

page.locator(selector,[options])

//Usando cualquier propiedad del cobjeto 
await page.click('id=user-name')
await page.locator('id=user-name').click();

//Usando CSS selector
await page.locator('#login-button').click();

//Usando XPATH(Muy importante)
await page.locator('xpath=//input[@name='login-button']').click();
await page.locator('//input[@name="login-button"]').click();
await page.locator('@id=user-name').click();

//Usando Text
await page.locator('text=LOGIN').click();
await page.locator('input:has-text("LOGIN")').click();
